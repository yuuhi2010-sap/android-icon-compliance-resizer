# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-06-08

- Initial public release.
- Add Android adaptive icon packing from existing artwork.
- Add Google Play Store `512x512` icon generation.
- Add launcher mask previews for circle, rounded-square, squircle, square, and safe-zone overlay.
- Add validation checks for generated resources and crop-risk indicators.
- Add beginner-focused README examples using real icon assets.
